# Oracle Installers Directory

This directory should contain the following files:

## Required Files

### SQLcl (Oracle SQL Command Line Interface)
- **File**: `sqlcl-25.2.2.199.0918.zip`
- **Download from**: [Oracle SQLcl Downloads](https://www.oracle.com/database/sqldeveloper/technologies/sqlcl/)
- **Description**: Oracle SQL Command Line Interface for database operations

## Installation Instructions

1. Download the required files from Oracle's official website
2. Place them in this directory
3. Ensure the filenames match exactly as specified above
4. Run the build process

## Notes

- These files are not included in the repository due to Oracle licensing requirements
- You must accept Oracle's license terms when downloading these files
- The build process will fail if these files are missing
