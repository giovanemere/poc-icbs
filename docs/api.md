# API Documentation

## 🔌 Endpoints

### GET /api/health
Verificación de salud del servicio.

**Response:**
```json
{
  "status": "ok",
  "timestamp": "2024-01-01T00:00:00Z"
}
```

### GET /api/version
Información de versión.

**Response:**
```json
{
  "version": "1.0.0",
  "build": "123"
}
```
