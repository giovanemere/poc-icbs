# Guía de Desarrollo

## 🛠️ Configuración del Entorno

### Prerrequisitos
- Node.js 18+
- Docker
- Git

### Instalación
```bash
git clone https://github.com/giovanemere/POC ICBS.git
cd POC ICBS
npm install
```

## 🧪 Testing

```bash
# Ejecutar tests
npm test

# Coverage
npm run test:coverage
```

## 📝 Contribución

1. Fork del repositorio
2. Crear rama feature
3. Commit cambios
4. Push a la rama
5. Crear Pull Request
