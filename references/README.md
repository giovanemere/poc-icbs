# Documentos de Referencia

Este directorio contiene documentos de referencia útiles para el proyecto Docker para Oracle WebLogic.

## Documentos disponibles

1. **[Comandos de WebLogic](./weblogic-commands.md)**
   - Comandos útiles para trabajar con Oracle WebLogic Server
   - Incluye comandos WLST, comandos de administración y comandos Docker

2. **[Integración de FF4J](./ff4j-integration.md)**
   - Guía para integrar FF4J (Feature Flipping For Java) en WebLogic
   - Incluye configuración de dependencias, web.xml y ejemplos de código

3. **[Despliegue Canary](./canary-deployment.md)**
   - Guía para implementar un despliegue canary en WebLogic
   - Incluye arquitectura, configuración y scripts de ejemplo

## Recursos externos

- [Documentación oficial de Oracle WebLogic](https://docs.oracle.com/middleware/12213/wls/index.html)
- [Documentación oficial de FF4J](https://ff4j.github.io/)

## Notas

Estos documentos son solo para referencia y pueden requerir adaptaciones según las necesidades específicas del proyecto. Asegúrese de revisar y probar cualquier configuración antes de aplicarla en un entorno de producción.
