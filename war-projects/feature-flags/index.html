🎨 Nuevo Diseño de UI

Habilita el nuevo diseño de interfaz de usuario (Versión B). Cuando está activo, redirige automáticamente a la versión B con las nuevas características visuales.

🚀 Versión B (Canary)

Activa la versión B como versión principal. Desactivado = versión A (estable). Esto controla qué versión se muestra por defecto a los usuarios.

🎯 A/B Testing

Habilita el testing A/B automático. Cuando está activo, el tráfico se distribuye entre las versiones A y B según la configuración del dashboard de tráfico.

🐦 Canary Deployment

Activa el despliegue canary. Permite dirigir un porcentaje controlado del tráfico a la nueva versión para validar su estabilidad antes del despliegue completo.
