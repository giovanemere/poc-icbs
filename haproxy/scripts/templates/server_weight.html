Pesos de Servidores

Los pesos de servidores determinan cómo se distribuye el tráfico entre los servidores de un mismo backend.

Valores de Peso
  • 0: El servidor no recibe tráfico nuevo, pero mantiene las conexiones existentes.
  • 1-255: Peso relativo para la distribución de tráfico.
  • 256: Peso máximo.
Recomendaciones
  • Utiliza pesos para balancear la carga según la capacidad de cada servidor.
  • Para deshabilitar temporalmente un servidor, establece su peso a 0.
  • Para realizar mantenimiento, reduce gradualmente el peso antes de detener el servidor.
